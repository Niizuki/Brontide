# Experimental and sideline projects

The following work is deliberately outside Linen Base conformance:

- `Linen.Experimental.Enrichment` explores targeted, explicit derivation outside Model and Kernel.
- `Linen.Experimental.Composition` explores Atlas Architecture 0.5 component selection, dependency
  gravity, boxed applications, support/opposition, explanations, and optimisation eligibility.
- Fabric/Linen entanglement is a future cross-process experiment. Neither implementation is allowed
  to become the other's in-process compatibility layer.

## GPU execution

GPU execution is both **experimental** and a **sideline project**. It is not implemented by the
current milestone path, is not required for Linen conformance, and cannot be counted as completion
of Composition, Imaging, or the mixed-stack workspace. CPU execution is the required reference
path. Any future GPU provider must declare eligibility, lowering failures, representation and copy
costs, fallback, interruption, and provider selection; it must remain replaceable without changing
the semantic image Operation.
