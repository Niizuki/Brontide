# `binding/portable/schemas/`

Data-only versioned neutral contracts, introduced in **PB1** (plan §5): canonical references and the
supported Shape floor; Component provisions and requirements; negotiated Operations, input/result/
detail Shapes and required Fragments; authority-presentation mode and cross-trust
`no-capability-transfer` declaration; inline and referenced-shaped-resource declarations;
delivery/hardening limits and lifecycle features; immutable Binding Plan facts; Channel envelopes,
correlation, protocol errors, and process-failure observations; and the C9 observation set.

No generated C#/F# source and no runtime helpers live here. If schemas generate code, generation runs
separately in each stack and this checked-in neutral source stays authoritative.

Empty until PB1.
