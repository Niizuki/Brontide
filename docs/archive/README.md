# Documentation archive

This directory preserves completed or superseded work. It is historical evidence, not a source of
new planned work.

## Foundation

Architecture 0.5 and everything earlier are grouped under [`foundation/`](./foundation/):

- Architecture 0.4 and 0.5;
- Minimal Stack Implementation Plan 0.2; and
- Reference Stack Implementation Plan 0.2.

- [`architecture/`](./architecture/) contains Architecture 0.6.

- [`corrections/`](./corrections/) contains the implementation-correction completion report.
